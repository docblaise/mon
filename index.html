Project Summary

This version keeps the interface simple: one summary area, one bindings table, one dropped-events table, and one raw log viewer. It is designed to sit on top of your Python script instead of replacing Scapy or the serial connection.

Recommended next step: have your Python script export a JSON file, then this page can fetch and display it automatically.
