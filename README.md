# Net-PY Demo Site

This folder contains a static GitHub Pages-ready demo site for Net-PY.

## Publish on GitHub Pages

1. Push this repository to GitHub.
2. In repository settings, open `Pages`.
3. Set the source to deploy from the `docs` folder on the default branch.
4. Save the settings and wait for GitHub Pages to publish the site.

## Files

- `index.html` - landing page and interactive demo shell
- `styles.css` - demo site styling
- `site.js` - sample data and section switching logic
